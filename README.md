# BildOro

Access the latest AI image generation models in one place.

## Setup

```bash
npm install
npm run db:generate
npm run dev
```

## Env Variables

```
DATABASE_URL=
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
SUPABASE_SERVICE_ROLE_KEY=
REPLICATE_API_TOKEN=
```

## Stack

-   Next.js 15 + TypeScript
-   Prisma + Supabase
-   Tailwind v4
-   Replicate AI
